# Generatore di Licenze ARSLink2

Questo è un'applicazione standalone per generare chiavi di licenza per il software ARSLink2.

## Istruzioni

1. Apri il file `index.html` in un browser web
2. Seleziona il tipo di licenza e la durata
3. Clicca su "Genera Licenza"
4. Copia la chiave di licenza generata
5. Fornisci la chiave al cliente

## Tipi di Licenza

- **Base**: Funzionalità di base senza integrazioni
- **Google Calendar**: Include sincronizzazione con Google Calendar
- **WhatsApp**: Include notifiche WhatsApp
- **Completa**: Include tutte le funzionalità

## Sicurezza

IMPORTANTE: Questo strumento deve essere utilizzato solo da personale autorizzato.
Non distribuire questo generatore ai clienti o a terze parti non autorizzate.

## Note Tecniche

Le licenze generate includono:
- Tipo di licenza
- ID univoco
- Data di scadenza codificata
- Checksum per la verifica dell'integrità

Le licenze generate vengono salvate localmente nel browser e possono essere copiate o eliminate secondo necessità.
